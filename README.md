# PC-Link Project

**Offline, cableless, Wi-Fi-less PC-to-PC communication system**

## Features
- Works without Wi-Fi, Bluetooth, or cables
- AI-enhanced signal decoding
- Short-range secure communication
- Can transfer messages, files, and commands
